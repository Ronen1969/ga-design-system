Como usar o Sistema de Marca

Três recursos disponíveis para toda a equipe. Siga os passos abaixo para configurar sua assinatura, emitir relatórios e preparar apresentações dentro do padrão Global Advising.

01
Prioridade imediata

Assinatura
de E-mail

  1. Acesse o gerador:
    https://ronen1969.github.io/ga-design-system/deliverables/
    mail-signature.html
  2. Preencha nome completo, cargo, departamento, telefone e e-mail.
  3. Clique em "Copiar como texto formatado".
  4. Abra as configurações de assinatura do seu cliente de e-mail.
  5. Cole com Ctrl+V e salve. Pronto.
02
Relatórios a clientes

Letterhead
para Relatórios

  1. Acesse o template:
    https://ronen1969.github.io/ga-design-system/deliverables/
    timbrado-word.html
  2. Substitua os dados de exemplo (cliente, data, referência GA-XXXX).
  3. Edite os títulos, texto e tabelas mantendo a estrutura existente.
  4. Pressione Ctrl+P → selecione "Salvar como PDF".
  5. Anexe o PDF ao e-mail para o cliente.
03
Decks e apresentações

Template de
Apresentação

  1. Acesse os slides:
    https://ronen1969.github.io/ga-design-system/deliverables/
    timbrado-powerpoint.html
  2. Navegue entre os 6 layouts com ← → ou os botões.
  3. Edite títulos, textos e dados diretamente no HTML — ou solicite ao time de design.
  4. Para exportar: Ctrl+P → "Salvar como PDF" (modo paisagem).
